# 3.1.2 SQL的特点

集**数据查询（data query）**、**数据操纵（data manipulation）**、**数据定义（data definition）**、**数据控制 （data control）**功能于一体。

### 主要特点

1. 综合统一
2. 高度非过程化
3. 面向集合的操作方式
4. 以同一种语法结构提供多种使用方式
5. 语言简洁，易学易用



### SQL的动词

| SQL功能 | 动词                     |
| ----- | ---------------------- |
| 数据查询  | SELECT                 |
| 数据定义  | CREATE, DROP, ALTER    |
| 数据操纵  | INSERT, UPDATE, DELETE |
| 数据控制  | GRANT, REVOKE          |
